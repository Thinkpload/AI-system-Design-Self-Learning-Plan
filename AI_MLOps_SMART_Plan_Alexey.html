Основа: программа Романа Юстина (EPAM) — AI System Design

Роман рекомендует подход top-down: сначала паттерны агентных систем → потом архитектура → потом конкретный стек. Ключевые инструменты: LangChain / LangGraph, CrewAI, Claude Agent SDK, Anthropic Cookbook. Книги: Chip Huyen (DMLS + AI Engineering), Valentina Alto, Alex Xu (System Design). Timeline Романа: Модули 1–4 (март–июль 2026) → Software Architecture (август+) → Tech Stack (осень 2026).

🐍 Python Foundations + Модуль 1: Core Agentic Patterns

March 15 – April 15, 2026 · Сейчас активно
🔵 Active Now ▼
📋 Roman — Модуль 1: Foundations of Agentic AI Systems (3–4 недели)
SMART Goal: К 15 апреля — написать первый рабочий AI-агент с минимум 2 паттернами от Романа (Prompt Chaining + Tool Use), запушить на GitHub с flowchart-схемой.

✅ Tasks

  • Настроить Python-окружение (VSCode, pip, venv)
  • Python basics: переменные, функции, циклы, классы
  • Git + GitHub: commit, push, branches, README
  • 1.1 Prompt Chaining — цепочка промптов с OpenAI/Claude API
  • 1.2 Routing — классификация запросов, маршрутизация
  • 1.3 Parallelization — async Python, fan-out/fan-in паттерны
  • 1.4 Reflection — self-critique и итеративное улучшение агента
  • 1.5 Tool Use — function calling, подключить внешний API
  • 1.6 Planning — декомпозиция задачи на подзадачи
  • 1.7 Multi-Agent — 2 агента с разными ролями
  • Flowchart: схема агентного пайплайна (Miro / draw.io)
  • Разобрать 2–3 примера в Anthropic Cookbook

🧠 Модуль 2: Adaptive & Contextual Intelligence

April 15 – May 15, 2026 · Month 2
⚪ Upcoming ▼
📋 Roman — Модуль 2: Adaptive and Contextual Intelligence (3–4 недели)
SMART Goal: К 15 мая — реализовать агента с долгосрочной памятью между сессиями, используя MCP-шаблоны от Романа. Агент «помнит» пользователя и адаптируется.

✅ Tasks

  • 2.1 Memory: short-term (сессия) vs. long-term (между сессиями)
  • Реализовать in-session память (контекст в диалоге)
  • Реализовать cross-session память (сохранение в БД/файл)
  • Сжатие контекста: summary-стратегии для длинных диалогов
  • 2.2 Learning & Adaptation — feedback loops, адаптивные промпты
  • 2.3 MCP — изучить протокол, написать свой шаблон
  • LangGraph: stateful агент с персистентным состоянием
  • 2.4 Goal Setting & Monitoring — динамическое целеполагание
  • Начать «AI Engineering» — Chip Huyen (часть 1)

3

🛡️ Модуль 3: Robustness, Human-in-the-Loop & RAG

May 15 – June 15, 2026 · Month 3
⚪ Upcoming ▼
📋 Roman — Модуль 3: Robustness, Human Collaboration & Knowledge Integration (3–4 недели)
SMART Goal: К 15 июня — построить RAG-систему на документах стартапа, добавить HITL-checkpoint с одобрением человека, симулировать 3 сценария отказа с graceful recovery.

✅ Tasks

  • 3.1 Exception Handling — retry-стратегии, fallback-цепочки
  • Graceful degradation: агент адаптируется, не падает
  • Симуляция 3 сценариев отказа (rate limit, timeout, bad output)
  • 3.2 Human-in-the-Loop (HITL) — точки одобрения в пайплайне
  • Построить HITL-checkpoint: агент запрашивает подтверждение
  • 3.3 RAG: векторная БД (Chroma / FAISS), индексирование
  • Семантический поиск по документам стартапа
  • Live RAG-демо: агент отвечает на вопросы по вашим docs
  • Читать «Designing ML Systems» — Chip Huyen (гл. 1–4)

⚡ Модуль 4: Advanced Architectures & Optimization

June 15 – July 15, 2026 · Month 4
⚪ Upcoming ▼
📋 Roman — Модуль 4: Advanced Agentic Architectures & Optimization (4–5 недель)
SMART Goal: К 15 июля — multi-agent система (≥2 агентов с A2A), Chain-of-Thought + Guardrails настроены, eval-dashboard с метриками качества работает.

✅ Tasks

  • 4.1 A2A Communication — message passing, shared state, blackboard
  • CrewAI: crew из 2+ агентов с разными ролями и задачами
  • 4.2 Resource-Aware Optimization — бюджет токенов, стоимость API

  • 4.3 Reasoning: Chain-of-Thought, ReAct, Tree-of-Thought
  • 4.4 Guardrails & Safety — фильтрация, constitutional constraints
  • 4.5 Evaluation & Monitoring — метрики, A/B тесты для AI
  • 4.6 Prioritization — управление очередью задач агента
  • 4.7 Exploration & Discovery — автономное расширение возможностей

🐳 Deploy: Docker, FastAPI, Cloud (MLOps-слой)

July 15 – August 15, 2026 · Month 5

📋 Stage 2 preview (Роман: август+) + практический DevOps/MLOps для агентных систем
