AI System Design: Core agentic patterns (Foundations of Agentic AI Systems) 
Goals:

Introduce core agentic patterns and their code implementations.
Build understanding of how agents process, route, and parallelize tasks.
Develop skills in chaining prompts and using tools for complex workflows.

Topics:
1. Prompt Chaining
2. Routing
3. Parallelization
4. Reflection
5. Tool Use
6. Planning
7. Multi-Agent
Case Studies: Integrate real-world case studies for each pattern.
Interactive Labs: Provide hands-on coding exercises after each chapter.
Visualization: Use flowcharts to illustrate agent workflows and decision points.




AI System Design: Adaptive and Contextual Intelligence 
Goals:

Teach how agents manage state, learn from experience, and adapt to new data.
Explore protocols for context management and dynamic goal tracking.

Topics:

1. Memory Management
2. Learning and Adaptation
3. Model Context Protocol (MCP)
4. Goal Setting and Monitoring

Memory Models: Compare short-term vs. long-term memory strategies.
Adaptive Loops: Include exercises on implementing feedback loops.
Protocol Templates: Provide reusable MCP templates for students.




AI System Design: Robustness, Human Collaboration, and Knowledge Integration 
Goals:

Ensure agents are robust to errors and can recover gracefully.
Highlight the importance of human oversight and knowledge integration.

Topics:
1. Exception Handling and Recovery
2. Human-in-the-Loop
3. Knowledge Retrieval (RAG)
Error Scenarios: Simulate failure cases and recovery strategies.
Human Feedback: Design assignments where students build HITL checkpoints.
RAG Demos: Use live demos to show retrieval-augmented generation in action.
