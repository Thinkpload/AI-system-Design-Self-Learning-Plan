🐍 Python & CS Foundations

March 15 – April 15, 2026 · Month 1
🔵 Active Now ▼
SMART Goal: By April 15, write a working Python script that uses a simple ML model (e.g., linear regression on a dataset), push it to GitHub, and understand Git version control basics.

✅ Weekly Tasks

  • Week 1: Set up Python environment (VSCode, pip, venv)
  • Week 1: Complete Python basics — variables, loops, functions
  • Week 2: Learn Git & GitHub — commit, push, branches

  • Week 4: Build mini project — Python script that reads CSV, cleans data, outputs stats
  • Week 4: Push project to GitHub with a README
